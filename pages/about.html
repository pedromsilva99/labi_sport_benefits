Acerca deste projeto

Autores:

  • José Pedro Silva - 89195 - jose.silva99@ua.pt
  • Pedro Miguel Silva - 89228 - pedromsilva99@ua.pt

Realizamos este trabalho para a disciplina de Laboratórios de Informática lecionada pelo professor António Adrego no âmbito de desenvolvermos as nossas capacidades de trabalho com HTML, JavaScript e CSS.

O José Silva limpou o template usado, criou e desenvolveu as páginas relacionadas com as vantagens económicas e o entretenimento, acrescentou o mapa e acrescentou o gráfico.

O Pedro Silva criou e desenvolveu a página inicial, as páginas relacionadas com as vantagens para a saúde, selecionou as imagens e acrescentou as notícias.

Assim a avaliação deste projeto deve ser dividida em 50% pelo dois membros do grupo.
